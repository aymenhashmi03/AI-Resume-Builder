# AI Resume Builder 

An **AI-powered Resume Builder web application** built with **React** that helps users create professional resumes quickly and efficiently. The platform provides a clean UI, dynamic form handling, and AI-assisted suggestions to improve resume quality.

---

## Features

- **AI-Powered Resume Suggestions**
- Dynamic Resume Form (Personal Info, Education, Experience, Skills)
- Modern & Responsive UI
- Real-time Resume Preview
- Download Resume as PDF
- Secure & Scalable Frontend Architecture
- Fast performance with optimized React components
